Webbens Historia

World Wide Web (WWW) skapades av Tim Berners Lee, en brittisk ingenjör och forskare som publicerades den 6 Augusti 1991

Berners-Lee arbetade vid CERN (Europiska organisationen för kärnforskning) när han utvecklade denna teknologi


HTTP är ett protokoll för att överföra data på webben

URL är webbadresser som används för att hitta sidor

DNS översätter webadresser till IP-adresser så att datorer kan hitta varandra


Netscape Navigator var den första populära webläsaren. Den blev populär 1990-talet men förlorade marknadsandelar till Internet Explorer och är numera inte i bruk.


Sveriges första hemsida skapades av Ulf Mårtensson och lanserades 1993. Hemsidan var en enkel sida som beskrev den svenska internetmiljön och teknlologin vid den tiden.
